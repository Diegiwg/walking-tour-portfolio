# walking-tour-portfolio

## Metas do Projecto

- Me ensinar um bom workflow de desenvolvimento para o web
- Criar um static webpage em que:
  - Exista `n` quadras de conteudo (por enquanto podem ser imagens, mas seria bom se pudessem ser qualquer elemento/tag html)
  - Uma figura que, ao clicar em uma quadra, faz um pathfind até a quadra clicada, e tenha um speech bubble de explicação sobre a quadra

## Visão

A ideia maior é para ter uma base de um site de portfolio que eu possa usar para mostrar meus projetos.
